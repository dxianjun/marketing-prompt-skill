# Marketing Prompt Skill

> 专业营销方案生成与引导助手 Skill

## 简介

这是一个基于 AI Agent 的营销方案生成技能。当用户需要生成营销方案但信息不完整时，通过结构化提问和示例引导用户补充完整的品牌背景、产品信息、目标用户、营销目标等关键信息，最终输出符合4A标准的专业营销活动方案。

## 功能特性

- 🎯 **智能引导**：信息不完整时，通过结构化提问逐步引导用户补充
- 💡 **示例驱动**：每个问题都附带示例回答，降低用户回答门槛
- 📋 **专业输出**：按照4A广告公司标准，输出五段式营销方案
- 🔄 **灵活适配**：只针对缺失的信息提问，不重复询问已提供内容
- 🎨 **多方向创意**：可生成多套完全不同方向的营销方案

## 适用场景

- 品牌营销策划
- 活动创意构思
- 推广方案制定
- 营销方案撰写
- 新品上市推广
- 节日营销策划
- 品牌传播策略
- 广告创意构思
- 整合营销方案
- 线上线下活动策划

## 工作流程

### 第一步：信息完整性判断

自动检查用户是否提供了完整的两大模块信息：

**模块一：背景定位信息**
- 品牌信息（品牌名称、核心调性、价值主张）
- 产品信息（品类、核心卖点）
- 用户画像（目标人群、年龄、圈层、消费习惯、偏好）

**模块二：目标明确信息**
- 执行动作（构思、撰写、分析等）
- 交付成果类型（创意点子、完整营销方案等）
- 方案数量（需要几套方案供选择）
- 时间/节点（营销季节、活动节点）

### 第二步：引导式提问补充

如果用户信息不完整，分模块进行提问引导：

**背景定位提问示例：**
> 为了帮你产出更精准的营销方案，我需要先了解一些背景信息：
>
> **1. 品牌信息**：你们的品牌叫什么？核心调性和价值主张是什么？
>    *示例：我们是一个主打"自然、活力、年轻"的新兴饮料品牌*
>
> **2. 产品信息**：产品属于什么品类？核心卖点是什么？
>    *示例：我们做的是气泡水，主打0糖0脂和真果汁添加*
>
> **3. 用户画像**：主要服务的人群是谁？（年龄、圈层、消费习惯等）
>    *示例：核心目标用户是Z世代和都市年轻白领，追求健康又好喝的饮品*

### 第三步：生成营销方案

信息完整后，按照五段式结构输出专业营销方案：

1. **主题名称**：给活动起一个易于传播、有记忆点的名字
2. **核心口号**：用最短的语言，戳中用户最深的渴望
3. **用户洞察**：阐述创意基于对目标用户何种心理或行为的洞察
4. **创意故事**：详细描述创意的核心内容，以及它如何展开
5. **渠道建议**：建议主要在哪些渠道进行传播

## 使用示例

### 示例1：信息不完整的情况

**用户输入**：
```
帮我做个营销方案
```

**技能响应**：
自动判断信息不完整，开始引导提问...

### 示例2：信息完整的情况

**用户输入**：
```
我们是一个主打"自然、活力、年轻"的新兴饮料品牌，做的是气泡水，主打0糖0脂和真果汁添加，核心目标用户是Z世代和都市年轻白领。

帮我构思3套完全不同方向的夏季营销方案。
```

**技能响应**：
直接生成3套不同方向的专业营销方案...
